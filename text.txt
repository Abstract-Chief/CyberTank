Для решения этой задачи я решил написать игру которая будет соответствовать большиству пунктов условий.
этой игрой стала игра про космический корабль который уворачивается от астероидов и может их сбивать.
Основными сложностями при реализации игры стали:
   генерация кубов с помощью генератора случайных чисел.
   управление игроком с помощью клавиатуры.
   система настроек параметров игры.
   движение кубов и их сбитие.
1 для решения первой проблемы я использую функцию get_rand(int min,int max)
 она генерерует случайное число от min до max с помощью функции rand();
2 управление игроком я осуществил через оператор switch который проверяет какую клавишу нажал пользователь через getch()
и делает соответствующее действие.
3 параметры игры я сделал с помощью глобальной переменной SETTING_ARR это массив в котором хранятся все настройки и их можно изменять через меню
4 в движении кубов используется функция gravity_cube()  которая вызывается каждый игровой цикл и опускает куб на один блок в низ если же он достигает дна он удаляется;
графическое оформление игры осуществялется функциями
   void graphick_info(); -- вывод информации
   void graphick_player(); -- вывод игрока
   void graphick_block(int x,int y); -- вывод одного блока
   void graphick(); -- вывод всех блококв
   void start_menu_graphick(int cur); -- вывод меню интерфейса
